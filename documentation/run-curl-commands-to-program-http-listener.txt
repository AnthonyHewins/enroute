Create a proxy
--------------
curl -X POST -H "Content-Type: application/json" -d '{"Name":"test"}'  http://localhost:1323/proxy
curl http://localhost:1323/proxy | python -m json.tool

Create a service
----------------
curl -X POST -H "Content-Type: application/json" -d '{"Service_Name":"test", "Fqdn":"testfqdn.com"}' http://localhost:1323/service | python -m json.tool
curl -X POST -H "Content-Type: application/json" -d '{"Service_name":"test"}' http://localhost:1323/proxy/test/service/test | python -m json.tool

Create a route for service
--------------------------
curl -X POST -H "Content-Type: application/json" -d '{"Route_name":"test", "Route_prefix":"/"}' http://localhost:1323/service/test/route | python -m json.tool

Delete an upstream
------------------
curl -X DELETE -H "Content-Type: application/json" -d '{"Upstream_name":"test", "Upstream_ip":"127.0.0.1", "Upstream_port":"9001"}' http://localhost:1323/upstream/test | python -m json.tool

Create an upstream
------------------
curl -X POST -H "Content-Type: application/json" -d '{"Upstream_name":"test", "Upstream_ip":"127.0.0.1", "Upstream_port":"9001", "Upstream_hc_path":"/", "Upstream_hc_host":"127.0.0.1"}' http://localhost:1323/upstream | python -m json.tool

Associate an upstream to a route
--------------------------------
curl -X POST http://localhost:1323/service/test/route/test/upstream/test | python -m json.tool
